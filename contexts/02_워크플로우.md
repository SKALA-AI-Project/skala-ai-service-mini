# 워크플로우

## 목표

웹 검색 기반 Agentic Workflow로 HBM4, PIM, CXL의 최신 정보를 수집하고 경쟁사별 성숙도와 위협 수준을 분석해 보고서를 자동 생성한다.

## 단계

1. 키워드와 경쟁사 조합으로 검색 쿼리 생성
2. 다중 관점 웹 검색 및 편향 점검
3. TRL 기반 기술 성숙도 분석
4. 7개 섹션 보고서 초안 생성
5. Markdown을 PDF로 변환

## 성공 기준

- 3개 기술 모두 포함
- 삼성전자와 마이크론 비교 포함
- 최근 3개월 이내 자료 포함
- 기술·기업별 TRL 명시
- 7개 섹션 구조 완비
- 출처 URL 추적 가능

## 제어 규칙

- Supervisor가 전체 순서를 통제한다.
- 검색, 초안, 포맷팅은 순차 실행한다.
- 단계별 재시도는 최대 2회다.
- 검색 다양성 미달이 2회 지속되면 HITL로 넘긴다.
