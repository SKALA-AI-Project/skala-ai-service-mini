# TRL 방법론

## 기본 원칙

- TRL 1~3: 논문·특허 기반 확인
- TRL 4~6: 간접지표 기반 추정
- TRL 7~9: 양산·출하·공식 발표 기반 확인

## 간접지표

- 특허 출원 패턴
- 학회 발표 빈도와 구체성
- 채용 공고 키워드

## confidence 규칙

- High: 지표 3개 모두 같은 방향
- Medium: 지표 2개가 같은 방향
- Low: 지표 충돌 또는 1개 이하 확인

## 서술 규칙

- `basis=confirmed`: 확인됨으로 작성
- `basis=estimated`: 추정됨으로 작성하고 한계 포함
- 공개가 보수적인 기업은 confidence를 보수적으로 설정
